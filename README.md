## Installations Guide
### Backend Commands to run
``` cd backend ```
``` npm install express mongoose cors ```
``` npm start ```

### Frontend Commands to run

``` cd frontend ```
``` npm init vue@latest ```
pick router when prompted but skip every other optional and the example code
``` npm run dev ```
